# Urban Thread

Urban Thread is a modern e-commerce frontend application built using React, Redux Toolkit, SCSS Modules, and Webpack.

The application allows users to:

- Browse t-shirts
- Filter products by size
- Sort products by price
- Add products to cart
- Update cart quantity
- View subtotal dynamically

---

# Tech Stack

- React
- Redux Toolkit (RTK)
- SCSS Modules
- Webpack
- Vitest

---

# Features

## Product Listing

- Product image
- Product title
- Product price
- Free shipping badge
- Responsive product grid

## Filters

- Multiple size selection
- Price sorting:
  - Low to High
  - High to Low

## Cart Sidebar

- Add to cart
- Remove from cart
- Quantity handling
- Dynamic subtotal calculation

## UI Features

- Skeleton loaders
- Empty states
- Responsive design
- Fixed sidebar layout

## Testing

- Unit Testing with Vitest

---
